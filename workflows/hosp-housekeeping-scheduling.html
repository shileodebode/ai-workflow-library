Automation Workflow
Grand Continental · Housekeeping Pipeline · Monday 06:00
INTAKE
🏨PMS Checkout Data
Pulls all checkout records from the PMS at 6am. Returns room number, guest departure time, room type, and next arrival time — all in one API call.
→
⚡Priority Scheduler
Ranks all checkout rooms by next arrival time, VIP status, and room type. Suites and early arrivals go to the top. Algorithm processes 50 rooms in under 2 seconds.
ASSIGN
👷Zone Staff Assignment
Assigns rooms to housekeepers based on floor zone, current workload, and seniority. Amaka gets floors 2-3, Ngozi gets floors 4-5 — balanced, no overlap.
→
📲WA Task to Housekeeper
WhatsApp message sent to each housekeeper: room list in priority order, special notes (VIP, extra linen, maintenance issues). Ticked off as they complete each room.
TRACK
📊Status Update Tracker
Housekeeper replies "done" to WhatsApp — room status updates to "Inspection Due" in real-time Google Sheets dashboard. Supervisor notified immediately.
→
🔍Inspection Prompt
Supervisor receives checklist prompt via WhatsApp: linens, bathroom, minibar, AC. Marks pass/fail. Only passing rooms advance to "Ready" status.
→
🔔Front Desk Alert
Instant WhatsApp to front desk: "Room 214 is ready for check-in." Triggered the moment inspection passes. Average alert time: 8 seconds after approval.
REPORT
📈Occupancy Dashboard
Live Google Sheets dashboard shows all room statuses at a glance — green (ready), yellow (cleaning), red (blocked). Management sees full hotel status from any device.
